راهنمای اجرای ربات بر روی سرور شخصی:

1. پیش‌نیازها:
   - مطمئن شوید که Python 3.10 یا بالاتر روی سرور شما نصب است.
   - ابزار pip باید نصب باشد.

2. نصب و اجرا:
   - فایل زیپ را در سرور خود اکسترکت کنید.
   - ترمینال را در پوشه اکسترکت شده باز کنید.
   - دستور زیر را اجرا کنید:
     bash start.sh

3. تنظیمات:
   - فایل .env را با ویرایشگر متن باز کنید.
   - مقادیر RPC_URL (آدرس نود اتریوم)، EXECUTOR_PRIVATE_KEY (کلید خصوصی ولت) و سایر تنظیمات را وارد کنید.
   - فایل را ذخیره کنید.

نکته: برای اجرای دائم (حتی پس از بستن ترمینال)، می‌توانید از ابزارهایی مانند screen یا tmux یا systemd استفاده کنید.
مثال با screen:
screen -S mevbot
bash start.sh
(سپس برای خروج بدون بستن ربات، کلیدهای Ctrl+A و سپس D را بزنید)
